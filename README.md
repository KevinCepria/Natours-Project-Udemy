# March 2019 update 
Dockerized the project:

To continue developing with Docker
- Use command "docker-compose up --build" to start the project
- Open project on localhost:8080
- You are good to go

To continue without using docker
- Run "npm install"
- Run "npm run start"
- Open project on localhost:8080
- You are good to go
